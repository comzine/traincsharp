# Übung: Mein erstes C#-Programm - Persönliche Daten und Rentenberechnung

## Ziel

Entwickle dein erstes Konsolenprogramm in C#, das den Benutzer nach seinem Namen und Alter fragt, und dann berechnet, wie viele Jahre bis zur Rente verbleiben.

## Anforderungen

1. **Name abfragen**: Fordere den Benutzer auf, seinen Namen einzugeben, und lies die Eingabe ein.

2. **Begrüßung ausgeben**: Gib eine persönliche Begrüßung mit dem eingegebenen Namen aus.

3. **Alter abfragen**: Fordere den Benutzer auf, sein Alter in Jahren einzugeben, und lies die Eingabe ein.

4. **Typkonvertierung**: Wandle die Alterseingabe (Text) in eine Zahl um, damit du damit rechnen kannst.

5. **Berechnung durchführen**: Berechne, wie viele Jahre bis zum Rentenalter (67 Jahre) verbleiben.

6. **Ergebnis ausgeben**: Gib dem Benutzer eine Nachricht aus, die seinen Namen und die berechneten Jahre bis zur Rente enthält.

## Optional

- Gib zusätzlich aus, in welchem Jahr der Benutzer in Rente gehen wird.
- Berechne und zeige an, wie viele Monate oder Tage bis zur Rente verbleiben.
- Formatiere die Ausgabe schöner, z.B. mit Trennlinien oder besonderen Formatierungen.
